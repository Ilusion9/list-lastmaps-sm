# Description
Informations about the last maps played. The informations are saved into lastmaps.ini.

# Alliedmods
https://forums.alliedmods.net/showthread.php?p=2686274

# Commands
```
sm_lastmaps
```

# ConVars
```
sm_lastmaps_maxsize 15 // How many maps will be shown in the map history list?
```
